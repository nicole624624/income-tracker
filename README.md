# 进账动力账本维护说明

这是一个手机优先的个人进账记录工具。它的核心目标不是做复杂财务系统，而是让每一次成交都被看见，形成继续创业的动力。

## 当前线上地址

- GitHub 仓库：`git@github.com:nicole624624/income-tracker.git`
- GitHub Pages：`https://nicole624624.github.io/income-tracker/`
- 本地预览常用地址：`http://127.0.0.1:4173/`

## 主要文件

- `index.html`：页面结构。顶部信念提醒、首页数字、目标水位、快速记账、更多功能卡片都在这里。
- `styles.css`：所有视觉样式。多巴胺渐变、玻璃质感、日历圆点、目标水位、手机布局都在这里调。
- `app.js`：页面交互。点击记账、切换日期、保存目标、每天轮换语录、飞书同步入口都在这里。
- `app-core.mjs`：纯计算逻辑。生成记账记录、计算月收入、月目标进度、每日/每月图表数据、飞书字段转换都在这里。
- `tests/`：自动测试。以后改功能前后都跑一遍，避免把已经可用的地方改坏。

## 当前功能

- 默认一单 `199`，首页可一键记入。
- 可以点日历圆点切换日期，补录以前的进账。
- 可以填写多单、例外金额和备注。
- 右上角设置可以修改默认单价和月目标。
- 首页有月进账、月目标、还差、完成比例、成交单数。
- 目标水位用椭圆水位展示，越接近目标填充越高。
- 本月每天用圆点日历展示收入和笔数。
- 今年每月默认折叠，需要时点击查看。
- 本月记录、飞书同步配置放在横向滑动区域里。
- 首页顶部有信念提醒：
  `你记下的每一笔，都是一个真实的人，把改善健康的希望托付给了你。`
- 目标卡里的短语录每天自动轮换，不显示“今日语录”标题。
